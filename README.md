# Travilifa

A simple multi-page travel and hospitality website.

## Pages

- Home: index.html
- Rooms: rooms.html
- Offers: offers.html
- Reservations: reservations.html
- About: about.html
- Contact: contact.html

## Project Structure

- HTML pages: *.html
- Styles: style.css
- Scripts: script.js
- Images: images/

## Usage

Open index.html in your browser, or host the folder with any static file server.
